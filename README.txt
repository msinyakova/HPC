Файл solve_for_1.cpp содержит решения дла L=1 MPI программы. Для случа L=pi необходимо запускать solve_for_pi.cpp. 
На вход программам solve_for_1.cp и solve_for_pi.cpp необходимо подать 2 числа. Первый аргумент - разбиение сетки N, второй аргумент - число процессоров для вычисления.
Для гибридной версии программы использовать файл solve_with_omp.cpp. 
На вход программe solve_with_omp.cpp необходимо подать 3 числа. Первый аргумент - разбиение сетки N, второй аргумент - число процессоров для вычисления, третий - число нитей.
